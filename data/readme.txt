身份证标注格式说明：
[91, [{"Rt": 0, "path": "001200.jpg", "scale": 0.5, "pts": [[76, 558], [512, 578], [508, 855], [60, 851]], "mirror": false}, ......]


91 - 总文件数

Rt: 旋转标识
    0 -- 不旋转
    1 -- 顺时针旋转90
    2 -- 顺时针旋转180
    3 -- 顺时针旋转270

path: 文件名

scale: 缩放比例，保持长宽比

pts: 缩放后，身份证4个角点位置，排列顺序左上，右上，左下，右下

mirror: 图像是否翻转之后进行标注（部分图片是拍摄镜子中的图像）
